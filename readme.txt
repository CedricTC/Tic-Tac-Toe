# Tic-Tac-Toe Oyunu

Bu basit bir Python Tic-Tac-Toe oyunudur. Kullanıcılar X ve O işaretleri arasında seçim yapabilirler ve oyunu kazanmak için üçünü de yatay, dikey veya çapraz olarak hizalamak zorundadırlar.

## Kurulum

1. Bu projeyi bilgisayarınıza  indirin.
2. Python yüklü değilse [Python'un resmi web sitesinden](https://www.python.org/) indirip kurun.
3. Projenin bulunduğu dizine gidin.
4. Terminal veya komut istemcisinde `python tic_tac_toe.py` komutunu çalıştırarak oyunu başlatın.

## Nasıl Oynanır

- Oyuncular sırayla bir hücreye X veya O işareti yerleştirirler.
- Üç X veya üç O yatay, dikey veya çapraz olarak hizalandığında, oyunu kazanırlar.
- Eğer tüm hücreler doluysa ve kazanan yoksa, oyun berabere biter.
